Echtzeit-Abfahrten direkt auf deinem Home-Bildschirm.

iPhone installieren

  1. 1Öffne live-departures.app in Safari
  2. 2Tippe unten auf das Teilen-Symbol (Quadrat mit Pfeil nach oben)
  3. 3Wähle „Zum Home-Bildschirm"
  4. 4Tippe oben rechts auf „Hinzufügen" – fertig

Android installieren

  1. 1Öffne live-departures.app in Chrome oder Samsung Internet
  2. 2Tippe rechts oben auf ⋮
  3. 3Wähle „App installieren" oder „Zum Startbildschirm"
  4. 4Bestätigen – fertig

Installieren

  1. 1Öffne live-departures.app in deinem Browser
  2. 2Nutze die Option „Zum Home-Bildschirm" oder „App installieren" im Browser-Menü
Jetzt öffnen
